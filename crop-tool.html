Zine Crop Configuration Tool

Define how to split fullpage zine images into individual pages

Instructions:
  1. Select a zine from the dropdown
  2. Click on a page in the right panel to select it
  3. Drag on the image to create a crop region for that page
  4. Repeat for all pages (cover, spreads, back cover)
  5. Copy the generated configuration and paste it into your main script
